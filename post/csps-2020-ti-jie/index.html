T2:动物园

根据题意，显然有：当饲料清单上某一位为0时，不能再加入一种动物使得这一位变为1，其余的位置可以为0或1

令SSS为所有饲料清单上第PiP_iPi​为000的个数（注意要去重）

那么最终答案就是2k−S−n2^{k-S}-n2k−S−n（k,nk,nk,n即为题目中所给的k,nk,nk,n）

初始的饲料清单可以记录一个数F=0F=0F=0，则FFF xori=1naixor_{i=1}^na_ixori=1n​ai​的结果就是饲料清单的值(令结果为FFF)

判断第PiP_iPi​位是否为000，可以判断F∣(1ull<<pi)F|(1ull<<p_i)F∣(1ull<<pi​)是否等于FFF。(这里的111要用ullullull)

需要特别注意的是最kkk最大可以达到646464，所以答案一定要用unsignedunsignedunsigned longlonglong longlonglong保存，不然会爆！

然而太弱的我并没想到k=64,S=0,n=0k=64,S=0,n=0k=64,S=0,n=0这种情况，因为它的答案为2642^{64}264，ullullull并不能存下它，那么就只有特判了!

#include<bits/stdc++.h>
#define ll long long

{
	unsigned ll x=0,f=1;char ch=getchar();

	while(ch>='0'&&ch<='9'){x=x*10+ch-'0';ch=getchar();} 
	return x*f;
}
bool vis[1000];
unsigned ll n,m,c,k,F,q,p;

	freopen("zoo.in","r",stdin);

	for(unsigned ll i=1,a;i<=n;i++)a=read(),F=F|a;

	{
		p=read(),q=read();
		if((F|(1ull<<p))!=F&&!vis[p])k--,vis[p]=true;

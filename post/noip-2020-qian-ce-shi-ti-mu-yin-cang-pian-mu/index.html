Problem A

Description

给一个 nnn 个节点 mmm 条边的无向图, 有 kkk 轮操作, 每轮操作是选择尽量多的边删除,

如果有多种方案, 那么选择边权和最大的那个, 但是要求删除的边中不存在环.

对于每条边, 输出它在第几次操作被删除, 如果这条边最后都没有被删除那么输出 000.

Task

Input

第一行三个正整数 n, m, k.

接下来 m 行, 每行三个正整数 ui,vi,wiu_i,v_i,w_iui​,vi​,wi​,表示一条连接 uiu_iui​ 和 viv_ivi​,边权为 wiw_iwi​ 的边.

Output

共 m 行, 每行一个整数表示答案.

Sample

Input

3 5 2
1 2 3
1 2 1
2 3 4
2 3 6
1 3 2

Output

1
0
2
1
2

Explanation

第一轮操作是删除 (1, 2, 3),(2, 3, 6) 这两条边.

第二轮操作是删除 (1, 3, 2),(2, 3, 4) 这两条边.

Constraints

对于所有数据, 保证 2 ≤ n ≤ 1000, 1 ≤ m ≤ 3 × 10510^5105, 1 ≤ k ≤ 10410^4104, 1≤ui,vi≤n,1≤wi≤1091 ≤ u_i , v_i ≤ n,1 ≤w_i ≤ 10^91≤ui​,vi​≤n,1≤wi​≤109, 且所有边权互不相同.

• 子任务 1(30 分): 保证 k ≤ 10.

• 子任务 2(70 分): 无特殊限制.

Problem B

Description

有一个 n×nn × nn×n 的黑白棋盘. 你需要用一些操作将整个棋盘变成全黑.

一次操作首先选择一行 iii, 一列 jjj, 记 c1,c2c_1, c_2c1​,c2​, · · · , cnc_ncn​ 为 (iii, 1),(i,2),⋅⋅⋅,(i,n)(i, 2), · · · ,(i,n)(i,2),⋅⋅⋅,(i,n) 的颜色.

之后将 (1, j),(2, j), · · · ,(n, j) 的颜色对应涂成 c1,c2,⋅⋅⋅,cnc_1, c_2, · · · , c_nc1​,c2​,⋅⋅⋅,cn​.

求最少的操作次数. 如果无法成功, 输出 −1.

Task

Input

第一行一个正整数 nnn.

接下来 nnn 行, 每行一个长度为 nnn 的字符串, 第 iii 行第 jjj 个字符表示 (iii,jjj) 的颜色.

其中′.′'.'′.′ 表示白色, '#' 表示黑色. (不含引号)

Output

一行一个整数表示答案.

Sample

Input

2
#.
.#

Output

3

Explanation

三次操作选择的 (i,j)分别为(1,2),(1,1),(1,2).(i, j) 分别为 (1, 2),(1, 1),(1, 2).(i,j)分别为(1,2),(1,1),(1,2).

Constraints

对于所有数据, 满足 1≤n≤500.1 ≤ n ≤ 500.1≤n≤500.

• 子任务 1(40 分): 保证 n≤4.n ≤ 4.n≤4.

• 子任务 2(60 分): 无特殊限制.

Problem C

Description

给定两个数 n,mn, mn,m, 求有多少长度为 2m2m2m 的序列 AAA 满足 AAA 中每个元素都是 nnn 的因数, 且 AAA所有项的乘积不超过 nmn^mnm. 答案对 998244353998244353998244353 取模.

Task

Input

一行两个正整数 n,m.n, m.n,m.

Output

一行一个整数表示答案对 998244353998244353998244353 取模的值.

Sample

Input

6 3

Output

2248

Constraints

对于所有数据, 满足 1≤n≤109,1≤m≤1001 ≤ n ≤ 10^9, 1 ≤ m ≤ 1001≤n≤109,1≤m≤100.

• 子任务 1(20 分): 保证 n≤50,m=2n ≤ 50, m = 2n≤50,m=2.

• 子任务 2(20 分): 保证 n≤100,m=3n ≤ 100, m = 3n≤100,m=3.

• 子任务 3(60 分): 无特殊限制.

Problem D

Description

有一棵深度为 nnn 的满二叉树, 初始每个节点都没有标记.

维护一个 nnn 位二进制数 xxx, 它对应了树根到叶子的一条路径: xxx 从高到低的每个二进制位
表示下一步走向左儿子 (这一位是 000) 还是右儿子 (这一位是 111).

初始 x=0x = 0x=0. 有 mmm 次操作, 每次操作令 x:=(x+2ic)modx := (x + 2^c_i ) modx:=(x+2ic​)mod 2n2^n2n, 然后将树上这条路径的所有节点打上标记. 每次操作后, 你需要输出有标记的节点个数.

Task

Input

第一行两个正整数 n,mn, mn,m.

接下来一行 mmm 个整数 cic_ici​.

Output

共 mmm 行, 每行一个正整数表示答案.

Sample

Input

2 3
0 1 0

Output

3
5
6

Explanation

假设根标号为 1, iii 的左儿子编号为 2⋅i2·i2⋅i, 右儿子编号为 2⋅i+12·i + 12⋅i+1.

第一次操作后, x=1x = 1x=1, 访问过的节点为 1, 2, 5.

第二次操作后, x=3x = 3x=3, 访问过的节点为 1, 2, 3, 5, 7.

第三次操作后, x=0x = 0x=0, 访问过的节点为 1, 2, 3, 4, 5, 7.

Constraints

对于所有数据, 满足 1≤n,m≤1051 ≤ n, m ≤ 10^51≤n,m≤105, 0≤ci≤n−10 ≤ c_i ≤ n − 10≤ci​≤n−1.

• 子任务 1(30 分): 保证 n,m≤5000n, m ≤ 5000n,m≤5000.

• 子任务 2(70 分): 无特殊限制.
